**Game Description:**
Our game's purpose is to test the luck between two players with rounds of coin spins. It is designed as a two-player game, where one user will type the number of rounds they would like to play. Users will enter their guesses respectively, with one player being player 1 and the other player 2. Once the guesses are in, the coin will spin, and the player(s) with the right guess will earn a point. Players will have the opportunity to enter the same option, and each gets a point if it's right. Once the rounds are done, if both players receive the same number of points, they will be given the option of a sudden-death round. This round will only end when both players enter different options.


In order to keep the rounds entertaining, users will have the option to make the game more difficult. If they select the difficult round, the number of coins being spun and guessed will increase. For example, if they were guessing one coin, the difficult round will make them guess two coins, so they will have the chance of gaining more points if they guess all coins correctly.
